Pre-requisites for running the zip file (intall if not installed): flask, pandas, matplotlib, seaborn, IPython

To run the flask application you need to locate to the extracted folder in command line, then input command 'flask run' command. A hyperlink will be generated by the cmd, you can go ahead and open that hyperlink in any browser, the linked html page should run just fine.

Put the inputs in there respective places and press submit button to generate the report. Insure that the date falls between 30:01:2020 - 11:08:2021, esle error will be generated, since the record is from 30:01:2020 to 11:08:2021.
